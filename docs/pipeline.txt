// External connections
External Sources > Collect External Data

// Main cycle flow
Collect Internal Data > Process in Backend
Collect External Data > Process in Backend
Process in Backend > Send to LLM Framework

Send to LLM Framework > Analyze with AI
Analyze with AI > Task Router

// 1. Fast vs Slow Track Routing
Task Router > Identify Patterns: Strategy (Slow Track) [color: purple]
Task Router > Customer Care Strategy: Chat (Fast Track) [color: orange]

Identify Patterns > Pricing Strategy
Identify Patterns > Content Strategy

// Human Guidance Flow
Receive User Input > Pricing Strategy: Priority guidance [color: green]
Receive User Input > Content Strategy: Priority guidance [color: green]
Receive User Input > Customer Care Strategy: Context guidance [color: green]

// Action Proposals
Pricing Strategy > Send Proposals to Dashboard
Content Strategy > Send Proposals to Dashboard
Customer Care Strategy > Safety Guardrail (LLM)

// 2. Trustworthy LLM & Safety Guardrail (LLM) Logic
Safety Guardrail (LLM) > Auto Reply to Customers: High Confidence/ Safe [color: green]
Safety Guardrail (LLM) > Send Proposals to Dashboard: Low Confidence/ Flagged [color: orange]

// Human Approval Flow
Send Proposals to Dashboard > User Approval
User Approval > Execute on Platforms: Approved
User Approval > Receive User Input: Declined (Provide feedback) [color: red]

// 3. Feedback Loop for Refinement
Receive User Input > Send to LLM Framework: Re-evaluate with feedback [color: red]

// API Execution
Execute on Platforms > Shopee + TikTok Shop API: Update via API [color: green]

// Learning Flow
Execute on Platforms > Measure Performance
Auto Reply to Customers > Measure Performance

Measure Performance > Store in Vector Database
Store in Vector Database > Optimize Future Decisions

// Cycle back to Observe
Optimize Future Decisions > Collect Internal Data: Next cycle [color: blue, textSize: large]
